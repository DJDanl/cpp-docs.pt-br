# <a name="active-template-library-atl-tutorial"></a>Tutorial da Biblioteca de Modelo Ativa (ATL)

A ATL foi projetada para simplificar o processo de criação de controles leves, flexíveis e eficientes. Este tutorial orienta você pela criação de um controle ActiveX, demonstrando muitos conceitos básicos de ATL e COM.

Seguindo este tutorial, você aprenderá a adicionar um controle a um projeto ATL que desenha um círculo e um polígono preenchido. Em seguida, você adicionará uma propriedade para indicar quantos lados o polígono terá e criará o código de desenho para atualizar o controle quando a propriedade for alterada. O controle será exibido em uma página da Web usando algum VBScript para fazer com que ele responda a eventos.

O tutorial é dividido em sete etapas. Você deve executar cada etapa na ordem, já que as etapas posteriores dependem das tarefas concluídas anteriormente. Antes de começar, você deve confirmar que tem privilégios necessários para registrar um componente ActiveX em seu computador específico. Isso geralmente é apenas uma preocupação se você estiver executando o Visual Studio .NET em uma conexão de serviços de terminal.

- [Etapa 2: adicionando um controle ao seu projeto](adding-a-control-atl-tutorial-part-2.md)

- [Etapa 3. Adicionando uma propriedade ao seu controle](adding-a-property-to-the-control-atl-tutorial-part-3.md)

- [Etapa 4: alterando o código de desenho do seu controle](changing-the-drawing-code-atl-tutorial-part-4.md)

- [Etapa 5: adicionar um evento](adding-an-event-atl-tutorial-part-5.md)
