# <a name="arithmetic-overflow-left-shift-count-is-negative-or-greater-than-or-equal-to-the-operand-size-which-is-undefined-behavior"></a>Estouro de aritmética: A contagem de turnos esquerdos é negativa ou maior ou igual ao tamanho do operand que é um comportamento indefinido

Este aviso indica que a contagem de turnos é negativa ou maior ou igual ao número de bits do opere sendo deslocado, resultando em comportamento indefinido.
Nota: C4293 é uma verificação semelhante no compilador Microsoft C++.

## <a name="example"></a>Exemplo

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return (hi << 32) | lo; // C26252 here
}
```

Para corrigir este aviso, use o seguinte código:

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return ((unsigned __int64)hi << 32) | lo; // OK
}
```
