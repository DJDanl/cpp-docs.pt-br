# <a name="c26462-use_const_pointer_for_variable"></a>C26462 USE_CONST_POINTER_FOR_VARIABLE

O valor apontado por '% Variable% ' é atribuído apenas uma vez, marque-o como um ponteiro para `const` (con. 4). 

Ponteiros para variáveis cujos valores permanecem inalterados devem ser marcados como const.

## <a name="see-also"></a>Veja também
[Diretrizes principais do C++ con. 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).

## <a name="example"></a>Exemplo
```cpp
void useVal(int val);

void function1(int* ptr)
{
    int* p = ptr; // C26462, the value pointed to by p is unmodified
    ptr = nullptr;

    useVal(*p);
}
```
