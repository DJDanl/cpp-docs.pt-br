# <a name="c26473-no_identity_cast"></a>C26473 NO_IDENTITY_CAST

"Não lance entre os tipos de ponteiros onde o tipo de origem e o tipo de destino são os mesmos."

**C++ Diretrizes principais**: Tipo.1: Evite moldes

Esta regra ajuda a remover moldes desnecessários ou suspeitos. Obviamente, quando o tipo é convertido a si mesmo, tal conversão é ineficaz, mas o fato de que o gesso é usado pode indicar uma questão de design sutil ou um potencial de regressão se os tipos mudarem no futuro. É sempre mais seguro usar o menor número possível de moldes.

## <a name="remarks"></a>Comentários

- Esta regra é implementada para estática e reinterpretar moldes e verifica apenas tipos de ponteiros.

## <a name="example"></a>Exemplo

olha perigosamente genérico

```cpp
gsl::span<server> servers_;

template<class T>
server* resolve_server(T tag) noexcept {
    auto p = reinterpret_cast<server*>(tag); // C26473, also 26490 NO_REINTERPRET_CAST
    return p >= &(*servers_.begin()) && p < &(*servers_.end()) ? p : nullptr;
}

void promote(server *s, int index) noexcept {
    auto s0 = resolve_server(s);
    auto s1 = resolve_server(index);
   if (s0 && s1)
        std::swap(s0, s1);
}
```

procure perigosamente genérica - reformulada

```cpp
// ...
server* resolve_server(server *p) noexcept {
    return p >= &(*servers_.begin()) && p < &(*servers_.end()) ? p : nullptr;
}

server* resolve_server(ptrdiff_t i) noexcept {
    return !servers_.empty() && i >= 0 && i < servers_.size() ? &servers_[i] : nullptr;
}
// ...
```
