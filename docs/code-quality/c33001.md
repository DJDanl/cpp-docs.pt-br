# <a name="c33001"></a>C33001

> Aviso C33001: a variante ' var ' foi apagada quando não foi inicializada (expressão ' expr ')

Esse aviso é disparado quando uma variante não inicializada é passada em uma API como VariantClear que espera uma variante inicializada.

## <a name="example"></a>Exemplo

```cpp
#include <Windows.h>

HRESULT foo(bool some_condition)
{
    VARIANT var;

    if (some_condition)
    {
        //...
        VariantInit(&var);
        //...
    }

    VariantClear(&var);     // C33001
}
```

Esses avisos são corrigidos garantindo que VariantClear seja chamado apenas para uma variante inicializada corretamente:
```cpp
#include <Windows.h>

HRESULT foo(bool some_condition)
{
    VARIANT var;

    if (some_condition)
    {
        //...
        VariantInit(&var);
        //...
        VariantClear(&var);     // C33001
    }
}
```
