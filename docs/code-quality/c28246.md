# <a name="c28246"></a>C28246

> aviso C28246: a anotação para a função '\<Name > '-Parameter '\<o parâmetro > ' não corresponde ao tipo do parâmetro

Um operador de __deref foi aplicado a um tipo de não ponteiro ao criar uma anotação.
