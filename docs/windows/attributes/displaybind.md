# <a name="displaybind"></a>displaybind

Indica uma propriedade que deve ser exibida para o usuário como vinculável.

## <a name="syntax"></a>Sintaxe

```cpp
[displaybind]
```

## <a name="remarks"></a>Comentários

O atributo **displaybind** C++ tem a mesma funcionalidade que o atributo MIDL [displaybind](/windows/win32/Midl/displaybind) .

## <a name="example"></a>Exemplo

Consulte o exemplo de [ligável](bindable.md) para obter um exemplo de como usar **displaybind**.
