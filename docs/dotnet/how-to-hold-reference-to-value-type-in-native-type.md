# <a name="how-to-hold-reference-to-value-type-in-native-type"></a>Como manter a referência para tipo de valor em tipo nativo

Use `gcroot` no tipo Boxed para manter uma referência a um tipo de valor em um tipo nativo.

## <a name="example"></a>Exemplo

```cpp
// reference_to_value_in_native.cpp
// compile with: /clr
#using <mscorlib.dll>
#include <vcclr.h>

using namespace System;

public value struct V {
   String ^str;
};

class Native {
public:
   gcroot< V^ > v_handle;
};

int main() {
   Native native;
   V v;
   native.v_handle = v;
   native.v_handle->str = "Hello";
   Console::WriteLine("String in V: {0}", native.v_handle->str);
}
```

```Output
String in V: Hello
```
