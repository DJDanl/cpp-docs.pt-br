# <a name="compiler-warning-level-1-c4113"></a>Aviso do compilador (nível 1) C4113

' identifier1 ' difere em listas de parâmetros de ' identifier2 '

Um ponteiro de função é atribuído a outro ponteiro de função, mas as listas de parâmetros formais das funções não são aceitas. A atribuição é compilada sem modificação.
