# <a name="compiler-warning-level-1-c4166"></a>Aviso do compilador (nível 1) C4166

**Convenção de chamada inválida para construtor/destruidor**

Construtores e destruidores não podem ter convenções de chamada diferentes do padrão para a plataforma (exceto quando você especifica explicitamente **__clrcall**).
