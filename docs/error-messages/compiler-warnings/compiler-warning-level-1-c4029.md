# <a name="compiler-warning-level-1-c4029"></a>Aviso do compilador (nível 1) C4029

lista de parâmetros formais declarada diferente da definição

Os tipos de parâmetro formais na declaração de função não são aceitos com aqueles na definição de função. O compilador usa a lista de parâmetros da definição.
