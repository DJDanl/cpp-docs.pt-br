# <a name="compiler-warning-c4961"></a>Aviso do compilador C4961

Nenhum dado de perfil foi mesclado no ' arquivo. pgd ', otimizações guiadas por perfil desabilitadas

Não há dados de perfil (arquivos. PGC) disponíveis; portanto, as otimizações guiadas por perfil não podem ocorrer.
