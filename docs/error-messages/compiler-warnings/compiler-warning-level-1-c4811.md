# <a name="compiler-warning-level-1-c4811"></a>Aviso do compilador (nível 1) C4811

valor de pragma em conformidade (forScope, show) = = valor

Esse aviso é emitido quando você usa a opção **Mostrar** do pragma de [conformidade](../../preprocessor/conform.md) . *valor* é o valor de conformidade atual.
