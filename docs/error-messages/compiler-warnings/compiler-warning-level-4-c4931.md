# <a name="compiler-warning-level-4-c4931"></a>Aviso do compilador (nível 4) C4931

> Estamos supondo que a biblioteca de tipos foi criada para ponteiros de *número*de bits

Não foram fornecidas informações explícitas com o atributo **ptrsize** da diretiva [#import](../../preprocessor/hash-import-directive-cpp.md) ; o compilador concluiu que o tamanho do ponteiro da biblioteca de tipos é *número*.

Esse aviso está desativado por padrão. Consulte [os avisos do compilador que estão desativados por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.
