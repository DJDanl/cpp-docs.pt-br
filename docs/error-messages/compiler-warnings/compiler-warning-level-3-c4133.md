# <a name="compiler-warning-level-3-c4133"></a>Aviso do compilador (nível 3) C4133

' type ': tipos incompatíveis-de ' type1 ' para ' type2 '

Esse aviso pode ser causado pela tentativa de subtrair dois ponteiros de tipos diferentes.

Para evitar esse aviso, forneça uma conversão de tipo apropriada.
