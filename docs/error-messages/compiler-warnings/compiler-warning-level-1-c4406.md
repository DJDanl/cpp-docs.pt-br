# <a name="compiler-warning-level-1-c4406"></a>Aviso do compilador (nível 1) C4406

operando na diretiva ignorado

A diretiva não pega nenhum operando, mas foi especificado um operando.

O compilador ignorou o operando ou operandos fornecidos.
