# <a name="compiler-warning-level-1-c4179"></a>Aviso do compilador (nível 1) C4179

> '//* ': analisado como '/' e '/\*': confusão com comentários '//' padrão

__//\*__ é um delimitador de comentário incorreto. Em vez disso, use __//__ ou __/\*__ .
