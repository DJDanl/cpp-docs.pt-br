# <a name="compiler-warning-level-4-c4820"></a>Aviso do compilador (nível 4) C4820

preenchimento de bytes 'bytes' adicionado após construção de 'member_name'

O tipo e a ordem dos elementos fizeram com que o compilador adicionasse o preenchimento ao final de um struct. Confira [alinhar](../../cpp/align-cpp.md) para obter mais informações sobre o preenchimento em uma estrutura.

Esse aviso está desativado por padrão. Consulte [os avisos do compilador que estão desativados por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.

O exemplo a seguir gera C4820:

```cpp
// C4820.cpp
// compile with: /W4 /c
#pragma warning(default : 4820)

// Delete the following 4 lines to resolve.
__declspec(align(2)) struct MyStruct {
   char a;
   int i;   // C4820
};

// OK
#pragma pack(1)
__declspec(align(1)) struct MyStruct2 {
   char a;
   int i;
};
```
