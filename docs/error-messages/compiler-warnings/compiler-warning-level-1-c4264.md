# <a name="compiler-warning-level-1-c4264"></a>Aviso do compilador (nível 1) C4264

' virtual_function ': nenhuma substituição disponível para função membro virtual da base ' class '; a função está oculta

C4264 sempre é gerado após [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).

Esse aviso está desativado por padrão. Consulte [os avisos do compilador que estão desativados por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.
