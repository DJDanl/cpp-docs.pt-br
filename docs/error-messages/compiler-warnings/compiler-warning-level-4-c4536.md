# <a name="compiler-warning-level-4-c4536"></a>Aviso do compilador (nível 4) C4536

' nome do tipo ': o nome do tipo excede o limite de metadados de ' limit ' caracteres

Um nome de tipo seria truncado nos metadados se fosse um tipo gerenciado. Consulte [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) para obter mais informações.

Esse aviso está desativado por padrão. Consulte [os avisos do compilador que estão desativados por padrão](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para obter mais informações.
