# <a name="compiler-warning-level-1-c4679"></a>Aviso do compilador (nível 1) C4679

> '*Member*': não foi possível importar o membro

O compilador encontrou um constructo para o qual ele não pode dar suporte, que não pode ser importado de metadados.

Não tente usar a construção.
