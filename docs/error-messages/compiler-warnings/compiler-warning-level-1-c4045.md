# <a name="compiler-warning-level-1-c4045"></a>Aviso do compilador (nível 1) C4045

' array ': estouro nos limites da matriz

A matriz tem muitos inicializadores. Inicializadores extras são ignorados.

Verifique se os elementos da matriz e os inicializadores correspondem em tamanho e quantidade.
