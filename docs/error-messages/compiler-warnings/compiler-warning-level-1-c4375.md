# <a name="compiler-warning-level-1-c4375"></a>Aviso do compilador (nível 1) C4375

o método não público ' method2 ' não substitui ' method1 '

Um tipo que implementa outro tipo definiu um método de substituição, mas a substituição não era pública. Portanto, o método não substitui o método de tipo base.
