# <a name="compiler-warning-level-1-c4022"></a>Aviso do compilador (nível 1) C4022

' function ': incompatibilidade de ponteiro para o parâmetro real ' Number '

O tipo de ponteiro do parâmetro real difere do tipo de ponteiro do parâmetro formal correspondente. O parâmetro real é passado sem alteração.
