# <a name="compiler-warning-level-1-c4030"></a>Aviso do compilador (nível 1) C4030

**primeira lista de parâmetros formais maior que a segunda lista**

Uma função foi redeclarada com parâmetros formais diferentes. O compilador usa os parâmetros formais fornecidos na primeira declaração.
