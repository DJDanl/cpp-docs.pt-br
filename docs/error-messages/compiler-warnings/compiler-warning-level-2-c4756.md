# <a name="compiler-warning-level-2-c4756"></a>Aviso do compilador (nível 2) C4756

estouro na aritmética de constante

O compilador gerou uma exceção ao fazer aritmética constante durante a compilação.

O exemplo a seguir gera C4756:

```cpp
// C4756.cpp
// compile with: /W2 /Od
int main()
{
   float f = 1e100+1e100;   // C4756
}
```
