# <a name="compiler-warning-level-1-c4038"></a>Aviso do compilador (nível 1) C4038

' Modifier ': modificador de classe de ambiente inválido

Este modificador não pode ser usado para classes com atributos **DllImport** ou [dllexport](../../cpp/dllexport-dllimport.md) .
