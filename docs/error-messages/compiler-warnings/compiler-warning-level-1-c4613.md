# <a name="compiler-warning-level-1-c4613"></a>Compilador C4613 de aviso (nível 1)
'segmento': classe de segmento não pode ser alterado.  
  
 Você tentou criar um segmento com o mesmo nome de classe como um segmento usado pelo compilador. Nenhuma nova classe de segmento foi criado.
