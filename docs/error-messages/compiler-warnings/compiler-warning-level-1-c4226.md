# <a name="compiler-warning-level-1-c4226"></a>Aviso do compilador (nível 1) C4226

> extensão não padrão usada: "*palavra-chave*" é uma palavra-chave obsoleta

A versão atual do Visual C++ não usa essa palavra-chave.

Esse aviso é promovido automaticamente para um erro.
