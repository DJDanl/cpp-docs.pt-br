# <a name="fatal-error-c1077"></a>Erro fatal C1077

limite do compilador: não é possível ter mais do que o número de opções de linha de comando

O número de opções de linha de comando excede o limite interno.

Pode haver muitos símbolos definidos com [/d](../../build/reference/d-preprocessor-definitions.md). (Em vez disso, coloque as definições em um arquivo de cabeçalho.)
