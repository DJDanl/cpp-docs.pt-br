# <a name="fatal-error-c1121"></a>Erro fatal C1121

falha na chamada para CryptoAPI

O compilador fez uma chamada para a CryptoAPI e a chamada falhou. Reinstale o Visual Studio e possivelmente reinstale o sistema operacional.
