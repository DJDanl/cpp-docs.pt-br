# <a name="fatal-error-c1038"></a>Erro fatal C1038

limite do compilador: função: o estado do fluxo de controle é muito complexo; simplificar a função

A função tem mais Estados de fluxo de controle do que o compilador pode manipular. Simplifique o fluxo de controle ou divida a função em funções menores.
