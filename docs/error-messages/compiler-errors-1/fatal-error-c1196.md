# <a name="fatal-error-c1196"></a>Erro fatal C1196

' identifier ': identificador encontrado na biblioteca de tipos ' TypeLib ' não é um C++ identificador válido

Um dos identificadores em sua biblioteca de tipos não é um identificador C++ válido. A biblioteca de tipos não está disponível para uso com [#import](../../preprocessor/hash-import-directive-cpp.md).
