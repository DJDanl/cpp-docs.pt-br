# <a name="compiler-error-c2308"></a>Erro do compilador C2308

concatenando cadeias de caracteres incompatíveis

As cadeias de caracteres largos e não largos foram especificadas para concatenação. Não é possível concatenar uma cadeia de caracteres larga e uma cadeia de caracteres que não seja larga.
