# <a name="compiler-error-c2163"></a>Erro do compilador C2163

' function ': não disponível como uma função intrínseca

Um `intrinsic` ou `function` pragma lista uma função não disponível no formulário intrínseco. Por exemplo, determinados intrínsecos não estão disponíveis durante a compilação de um programa que usa programação/CLR.
