# <a name="fatal-error-c1202"></a>Erro fatal C1202

tipo recursivo ou contexto de dependência de função muito complexo

Uma definição de modelo foi recursiva ou excedeu os limites de complexidade.

## <a name="examples"></a>Exemplos

O exemplo a seguir gera C1202.

```cpp
// C1202.cpp
// processor: x86 IPF
template<int n>
class Factorial : public Factorial<n-1>  {   // C1202
public:
   operator int () {
      return Factorial <n-1>::operator int () * n;
   }
};
Factorial<7> facSeven;
```

Resolução possível.

```cpp
// C1202b.cpp
// compile with: /c
template<int n>
class Factorial : public Factorial<n-1> {
public:
   operator int () {
      return Factorial <n-1>::operator int () * n;
   }
};

template <>
class Factorial<0> {
public:
   operator int () {
      return 1;
   }
};

Factorial<7> facSeven;
```
