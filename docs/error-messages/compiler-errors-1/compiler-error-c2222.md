# <a name="compiler-error-c2222"></a>Erro do compilador C2222

tipo inesperado ' type ': uma classe base ou um membro era esperado

A lista de inicializadores só pode inicializar classes base ou membros de um tipo. Para corrigir esse erro, verifique se somente as classes base ou os membros do tipo são inicializados na lista de inicializadores.
