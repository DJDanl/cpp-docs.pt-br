# <a name="compiler-error-c2083"></a>Erro do compilador C2083

comparação de struct/Union inválida

Uma estrutura ou União é comparada diretamente com outro tipo definido pelo usuário. Isso não é permitido, a menos que um operador de comparação tenha sido definido ou uma conversão para um tipo escalar exista.
