# <a name="fatal-error-c1904"></a>Erro fatal C1904

interação com o provedor inadequada: ' file '

Esse erro indica a falha de um provedor de atributos.

Observe as circunstâncias do erro, tente isolar o problema e criar um caso de teste reproduzível e entre em contato [conosco](/visualstudio/ide/talk-to-us).
