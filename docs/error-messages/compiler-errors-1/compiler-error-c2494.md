# <a name="compiler-error-c2494"></a>Erro do compilador C2494

' palavra-chave ' não pode ser chamada de dentro de uma expressão de filtro ou __finally bloco/finally

Você não pode usar `keyword` em um bloco `__finally` ou finally.

O exemplo a seguir gera C2494:

```cpp
// C2494.cpp
#include <malloc.h>

int main() {
   __try {}
   __except ( _alloca(100), 1 ) {}   // C2494
   __try {}
   __finally {
      _alloca(100);   // C2494
   }
}
```

C2494 também pode ocorrer ao usar **/CLR**.

```cpp
// C2494b.cpp
// compile with: /clr
#include <malloc.h>

int main() {
   char * buf;
   try {}
   catch (char * buf2) {}
   finally {
      _alloca(100);   // C2494
   }
}
```
