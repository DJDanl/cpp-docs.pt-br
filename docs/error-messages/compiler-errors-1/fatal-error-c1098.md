# <a name="fatal-error-c1098"></a>Erro fatal C1098

Incompatibilidade de versão com o mecanismo editar e continuar

A versão do depurador que você está usando não corresponde ao compilador usado para criar o executável. Se a recompilação não corrigir o problema, talvez seja necessário reinstalar o C++ Visual para ter certeza de que você tem as versões apropriadas do depurador e do compilador.
