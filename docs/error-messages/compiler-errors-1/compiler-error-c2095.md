# <a name="compiler-error-c2095"></a>Erro do compilador C2095

' function ': o parâmetro real tem o tipo ' void ': parâmetro ' Number '

O parâmetro passado para a função é do tipo `void`, o que não é permitido. Use um ponteiro para void (`void *`) em vez disso.

O `number` indica qual parâmetro é `void`.
