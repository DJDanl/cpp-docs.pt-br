# <a name="compiler-error-c2048"></a>Erro do compilador C2048

mais de um padrão

Uma **`switch`** instrução contém vários **`default`** Rótulos. Exclua um dos **`default`** Rótulos para resolver o erro.

O exemplo a seguir gera C2048:

```cpp
// C2048.cpp
int main() {
   int a = 1;
   switch (a) {
      case 1:
         a = 0;
      default:
         a = 2;
      default:   // C2048
         a = 3;
   }
}
```

Resolução possível:

```cpp
// C2048b.cpp
int main() {
   int a = 1;
   switch (a) {
      case 1:
         a = 0;
      default:
         a = 2;
   }
}
```
