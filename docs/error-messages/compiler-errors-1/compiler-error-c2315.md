# <a name="compiler-error-c2315"></a>Erro do compilador C2315

' type1 ': a referência é capturada por ' type2 ' no número de linha

O tipo de exceção é tratado por um manipulador anterior. A referência para o segundo catch tem o tipo do primeiro.
