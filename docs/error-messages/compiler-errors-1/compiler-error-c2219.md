# <a name="compiler-error-c2219"></a>Erro do compilador C2219

erro de sintaxe: o qualificador de tipo deve ser posterior a ' * '

O qualificador de tipo ( **`const`** ou **`volatile`** ) é exibido onde não é permitido.
