# <a name="compiler-error-c2489"></a>Erro do compilador C2489

' identifier ': a variável automática ou de registro inicializada não é permitida no escopo de função na função ' naked '

Para obter mais informações, consulte [Naked](../../cpp/naked-cpp.md).

O exemplo a seguir gera C2489:

```cpp
// C2489.cpp
// processor: x86
__declspec( naked ) int func() {
   int i = 1;   // C2489
   register int j = 1;   // C2489
}
```
