# <a name="fatal-error-c1313"></a>Erro fatal C1313

limite do compilador: blocos de tipos não podem ser aninhados mais profundamente que os níveis de número

Blocos de tratamento de exceção (ou manipulação de exceção estruturada) estavam aninhados muito profundamente.  Simplifique seu código.
