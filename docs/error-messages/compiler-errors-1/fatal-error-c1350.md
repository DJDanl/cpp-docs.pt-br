# <a name="fatal-error-c1350"></a>Erro fatal C1350

erro ao carregar a dll ' dll ': dll não encontrada

A DLL que dá suporte à operação tentada não foi encontrada.

Isso indica um problema com a instalação e você deve reinstalar a DLL do CD do produto.
