# <a name="fatal-error-c1057"></a>Erro fatal C1057

fim de arquivo inesperado na expansão de macro

O compilador atingiu o final do arquivo de origem durante a coleta de argumentos de invocação de macro, provavelmente devido a um parêntese direito ausente na chamada de macro.
