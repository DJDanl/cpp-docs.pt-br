# <a name="compiler-error-c2317"></a>Erro do compilador C2317

o bloco ' Try ' começando na linha ' Number ' não tem manipuladores catch

Um bloco de `try` deve ter pelo menos um manipulador catch.

O exemplo a seguir gera C2317:

```cpp
// C2317.cpp
// compile with: /EHsc
#include <eh.h>
int main() {
   try {
      throw "throw an exception";
   }
   // C2317, no catch handler
}
```

Resolução possível:

```cpp
// C2317b.cpp
// compile with: /EHsc
#include <eh.h>
int main() {
   try {
      throw "throw an exception";
   }
   catch(char*) {}
}
```
