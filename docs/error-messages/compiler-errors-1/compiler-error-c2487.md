# <a name="compiler-error-c2487"></a>Erro do compilador C2487

' identifier ': membro da classe de interface DLL não pode ser declarado com interface DLL

Você pode declarar uma classe inteira ou determinados membros de uma classe de interface não-DLL, com a interface DLL. Você não pode declarar uma classe com a interface DLL e, em seguida, declarar um membro dessa classe com a interface DLL.
