# <a name="compiler-error-c2151"></a>Erro do compilador C2151

mais de um atributo de idioma

Uma função tem mais de uma palavra-chave ( **`__cdecl`** , **`__stdcall`** ou **`__fastcall`** ) especificando uma Convenção de chamada.
