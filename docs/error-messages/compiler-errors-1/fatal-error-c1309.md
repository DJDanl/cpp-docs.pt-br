# <a name="fatal-error-c1309"></a>Erro fatal C1309

Versões do C2 incompatíveis. DLL e PGODB\<versão >. DLL

O conjunto de ferramentas que você está usando para criar e usar [otimizações guiadas por perfil](../../build/profile-guided-optimizations.md) contém componentes incompatíveis.  Se você não puder resolver manualmente esse erro, reinstale o Visual C++.
