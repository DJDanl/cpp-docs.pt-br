# <a name="compiler-error-c2223"></a>Erro do compilador C2223

esquerda de '-> identificador ' deve apontar para struct/Union

O operando à esquerda de `->` não é um ponteiro para uma classe, estrutura ou União.

Esse erro pode ser causado por um operando esquerdo que é uma variável indefinida (portanto, digite `int`).
