# <a name="compiler-error-c2047"></a>Erro do compilador C2047

padrão inválido

A palavra-chave **`default`** pode aparecer apenas em uma **`switch`** instrução.

O exemplo a seguir gera C2047:

```cpp
// C2047.cpp
int main() {
   int i = 0;
   default:   // C2047
   switch(i) {
      case 0:
      break;
   }
}
```

Resolução possível:

```cpp
// C2047b.cpp
int main() {
   int i = 0;
   switch(i) {
      case 0:
      break;
      default:
      break;
   }
}
```
