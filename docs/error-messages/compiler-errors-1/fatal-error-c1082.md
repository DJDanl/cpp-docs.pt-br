# <a name="fatal-error-c1082"></a>Erro fatal C1082

Não é possível fechar o arquivo filetype: ' file ': Message

Se a mensagem disser "número de arquivo inadequado", o arquivo pode ter sido fechado em primeiro plano durante a compilação em segundo plano.
