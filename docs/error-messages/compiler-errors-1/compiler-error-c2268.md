# <a name="compiler-error-c2268"></a>Erro do compilador C2268

' function ' é um auxiliar de biblioteca predefinida do compilador. Não há suporte para auxiliares de biblioteca com/GL; Compilar arquivo de objeto ' file ' sem/GL.

Uma função definida no código-fonte tem o mesmo nome que uma função de compilador interna. Compile o módulo que contém a função sem [/GL](../../build/reference/gl-whole-program-optimization.md).

O exemplo a seguir gera C2268:

```c
// C2268.c
// compile with: /c
// processor: x86
extern int SHFusionLoadLibrary(int lpLibFileName);

int __cdecl _except_handler3(void) {
   return SHFusionLoadLibrary(0);
}

extern int main(void);

void* mainCRTStartup(void* p) {
   p = main;
   return p;
}
```

e, em seguida:

```c

```
