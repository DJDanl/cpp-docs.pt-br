# <a name="compiler-error-c2050"></a>Erro do compilador C2050

expressão switch não integral

A expressão `switch` é avaliada como um valor não inteiro. Para resolver o erro, use apenas valores integrais em instruções switch.

O exemplo a seguir gera C2050:

```cpp
// C2050.cpp
int main() {
   int a = 1;
   switch ("a") {   // C2050
      case 1:
         a = 0;
      default:
         a = 2;
   }
}
```

Resolução possível:

```cpp
// C2050b.cpp
int main() {
   int a = 1;
   switch (a) {
      case 1:
         a = 0;
      default:
         a = 2;
   }
}
```
