# <a name="fatal-error-c1045"></a>Erro fatal C1045

limite do compilador: especificações de vinculação aninhadas muito profundamente

As externas aninhadas excedem o limite do compilador. As externas aninhadas são permitidas com o tipo de ligação externa, comoC++`extern` "". Reduza o número de externas aninhadas para resolver o erro.
