# <a name="compiler-error-c2077"></a>Erro do compilador C2077

' identificador ' de inicializador de campo não escalar

Você tentou inicializar um campo de bits com um não escalar (struct, Union, matriz ou classe). Use um valor inteiro ou de ponto flutuante.
