# <a name="fatal-error-c1601"></a>Erro fatal C1601

opcode de assembly embutido sem suporte

Esse erro indica uma incompatibilidade nos arquivos do compilador. exe, que podem ter ocorrido devido a uma instalação incompleta. Por exemplo, você pode ter instalado um service pack, mas não o pacote de processador. Instale todos os produtos necessários.
