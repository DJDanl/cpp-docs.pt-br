# <a name="compiler-error-c2465"></a>Erro do compilador C2465

Não é possível definir um tipo anônimo dentro dos parênteses

Uma estrutura anônima, União ou tipo enumerado é definida dentro de uma expressão entre parênteses. Isso é inválido no C++ porque a definição não tem sentido no escopo da função.
