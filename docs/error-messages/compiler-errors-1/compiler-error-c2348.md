# <a name="compiler-error-c2348"></a>Erro do compilador C2348

' Type Name ': não é uma agregação em estilo C, não pode ser exportada em IDL inserida

Para posicionar um `struct` em um arquivo. idl com o atributo de [exportação](../../windows/export.md) , o `struct` deve conter apenas dados.

O exemplo a seguir gera C2348:

```cpp
// C2348.cpp
// C2348 error expected
[ module(name="SimpleMidlTest") ];

[export]
struct Point {
   // Delete the following two lines to resolve.
   Point() : m_i(0), m_j(0) {}
   Point(int i, int j) : m_i(i), m_j(j) {}

   int m_i;
   int m_j;
};
```
