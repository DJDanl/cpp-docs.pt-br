# <a name="compiler-error-c2301"></a>Erro do compilador C2301

esquerda de '-> ~ identifier ' deve apontar para Class/struct/Union

A expressão à esquerda do operador de `->` não é avaliada como um ponteiro para uma classe, estrutura ou União.
