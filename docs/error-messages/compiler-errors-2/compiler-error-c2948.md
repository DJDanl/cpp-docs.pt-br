# <a name="compiler-error-c2948"></a>Erro do compilador C2948

instanciação explícita; especificador de classe de armazenamento ' especificador ' não permitido em especialização

Você não pode usar especificadores de classe de armazenamento (como **`extern`** ) em uma especialização de uma classe de modelo que foi instanciada anteriormente explicitamente.
