# <a name="compiler-error-c2551"></a>Erro do compilador C2551

o tipo ' void * ' precisa de uma conversão explícita

Um **`void`** ponteiro é atribuído a um ponteiro não void por conversão implícita. Você deve usar uma conversão explícita.
