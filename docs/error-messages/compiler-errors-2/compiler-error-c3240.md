# <a name="compiler-error-c3240"></a>Erro do compilador C3240

' function ': deve ser uma função de membro abstract não sobrecarregada de ' type '

Um tipo base continha uma função que foi definida. A função deve ser virtual.

## <a name="example"></a>Exemplo

O exemplo a seguir gera C3240.

```cpp
// C3240.cpp
// compile with: /c
__interface I {
   void f();
};

struct A1 : I {
   void f() {}
};

struct A2 : I {
   void f() = 0;
};

template <class T>
struct A3 : T {
   void T::f() {}
};

template <class T>
struct A4 : T {
   void T::f() {}
};

A3<A1> x;   // C3240
A3<I> x2;
A4<A2> x3;
```
