# <a name="compiler-error-c3150"></a>Erro do compilador C3150

"element": "Attribute" pode ser aplicado somente a uma classe, interface, matriz ou ponteiro

**`__gc`** Só pode ser usado em uma classe, interface ou matriz.

C3150 só é acessível usando a opção de compilador obsoleto **/CLR: oldSyntax**.
