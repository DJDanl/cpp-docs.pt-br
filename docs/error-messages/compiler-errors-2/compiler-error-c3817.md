# <a name="compiler-error-c3817"></a>Erro do compilador C3817

' declaração ': a propriedade só pode ser aplicada a uma função

A **`property`** palavra-chave só pode ser aplicada a uma definição de função.

C3817 só é acessível usando a opção de compilador obsoleto **/CLR: oldSyntax**.
