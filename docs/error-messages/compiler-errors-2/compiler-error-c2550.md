# <a name="compiler-error-c2550"></a>Erro do compilador C2550

' identifier ': listas de inicializadores de Construtor são permitidas somente em definições de Construtor

Uma lista de inicializadores de classe base é usada na definição de uma função que não é um construtor.

O exemplo a seguir gera C2550:

```cpp
// C2550.cpp
// compile with: /c
class C {
public:
   C();
};

class D : public C {
public:
   D();
   void func();
};

void D::func() : C() {}  // C2550
D::D() : C() {}   // OK
```
