# <a name="compiler-error-c2573"></a>Erro do compilador C2573

' class ': não é possível excluir ponteiros para objetos deste tipo; a classe não tem sobrecarga de não posicionamento para ' operator Delete '.

A classe não tem um operador de exclusão sem posicionamento.
