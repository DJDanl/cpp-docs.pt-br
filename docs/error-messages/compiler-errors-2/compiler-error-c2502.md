# <a name="compiler-error-c2502"></a>C2502 de erro do compilador
'Identificador': muitos modificadores de acesso na classe base  
  
 A classe base tiver mais de um modificador de acesso. Modificador de acesso de somente um (`public`, `private`, ou `protected`) Ã© permitido.  
  
 O exemplo a seguir gera C2502:  
  
```  
// C2502.cpp  
// compile with: /c  
class A { };  
class B { };  
class C : private public A { };   // C2502  
  
// OK  
class D : private A {};  
class E : public A, private B {};  
```
