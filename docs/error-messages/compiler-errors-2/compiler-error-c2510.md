# <a name="compiler-error-c2510"></a>Erro do compilador C2510

' identifier ': esquerda de ':: ' deve ser uma classe/struct/Union

Um nome de classe, estrutura ou União deve aparecer no lado esquerdo do operador do operador de resolução de escopo (`::`).
