# <a name="compiler-error-c3130"></a>Erro do compilador C3130

Erro de compilador interno: falha ao gravar bloco de código injetado em PDB

Esse erro ocorrerá se o compilador falhar ao gravar um bloco de código injetado no arquivo. pdb. O motivo mais comum para a falha é a falta de espaço em disco.
