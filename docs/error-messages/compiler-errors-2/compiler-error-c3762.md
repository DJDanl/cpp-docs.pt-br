# <a name="compiler-error-c3762"></a>Erro do compilador C3762

Não é possível processar o atributo ' attribute '

Um atributo definido pelo usuário que herda de `System.Security.Permissions.SecurityAttribute` está sendo usado para definir um atributo de segurança. Esse atributo não pode ser usado no mesmo assembly em que está definido.
