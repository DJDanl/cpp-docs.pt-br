# <a name="compiler-error-c3115"></a>Erro do compilador C3115

' attribute ': Este atributo não é permitido em ' Construct '

Um atributo foi aplicado a um constructo para o qual ele não foi planejado.  Consulte [atributos por uso](../../windows/attributes/attributes-by-usage.md) para obter mais informações.

## <a name="example"></a>Exemplo

O exemplo a seguir gera C3115.

```cpp
// C3115.cpp
// compile with: /c
#include <unknwn.h>
[module(name="xx")];

[object, helpstringdll(xx.dll), uuid("00000000-0000-0000-0000-000000000001")]   // C3115
// try the following line instead
// [object, uuid("00000000-0000-0000-0000-000000000001")]
__interface IMyI {
   HRESULT xx();
};
```
