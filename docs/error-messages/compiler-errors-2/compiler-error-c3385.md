# <a name="compiler-error-c3385"></a>Erro do compilador C3385

' classe:: function ': uma função que tem um atributo personalizado DllImport não pode retornar uma instância de uma classe

Uma função definida como estando em um arquivo. dll especificado com o atributo `DllImport` não pode retornar uma instância de uma classe.

O exemplo a seguir gera C3385:

```cpp
// C3385.cpp
// compile with: /clr /c
using namespace System;
using namespace System::Runtime::InteropServices;

struct SomeStruct1 {};

public ref struct Wrap {
   [ DllImport("somedll.dll", CharSet=CharSet::Unicode) ]
   static SomeStruct1 f1([In, Out] SomeStruct1 *pS);   // C3385
};
```
