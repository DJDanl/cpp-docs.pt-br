# <a name="compiler-error-c2654"></a>Erro do compilador C2654

' identifier ': tentativa de acessar membro fora de uma função membro

Um membro é acessado em uma declaração. Os dados do membro podem ser acessados somente em funções de membro.

Esse erro pode ser causado ao tentar inicializar variáveis em uma declaração. Use um construtor para essa finalidade.
