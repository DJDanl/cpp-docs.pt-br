# <a name="compiler-error-c2569"></a>Erro do compilador C2569

' EnumOrUnion ': enum/Union não pode ser usado como uma classe base

Se você precisar derivar um tipo da União ou enumeração especificada, altere a União ou a enumeração para uma classe ou estrutura.

O exemplo a seguir gera C2569:

```cpp
// C2569.cpp
// compile with: /c
union ubase {};
class cHasPubUBase : public ubase {};   // C2569
// OK
struct sbase {};
class cHasPubUBase : public sbase {};
```
