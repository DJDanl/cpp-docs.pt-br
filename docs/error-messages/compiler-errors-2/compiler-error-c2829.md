# <a name="compiler-error-c2829"></a>Erro do compilador C2829

' operator Operator ' não pode ter uma lista de parâmetros de variável

Somente dois operadores podem ter listas de parâmetros variáveis:

- Chamada de função`()`

- **`new`**
