# <a name="compiler-error-c2855"></a>Erro do compilador C2855

opção de linha de comando ' Option ' inconsistente com cabeçalho pré-compilado

A opção de linha de comando fornecida difere da opção usada para criar o cabeçalho pré-compilado. Gere novamente o cabeçalho pré-compilado com a opção de linha de comando fornecida.
