# <a name="compiler-error-c3724"></a>Erro do compilador C3724

é necessário #include \<Windows. h > para usar multithreading com eventos

O arquivo Windows. h será necessário se você usar vários threads com eventos. Para corrigir esse erro, adicione `#include <windows.h>` à parte superior do arquivo no qual as fontes de eventos e os receptores de eventos são definidos.

```cpp
// C3724.cpp
// uncomment the following line to resolve
// #include <windows.h>

[event_source(native), threading(apartment)]
class CEventSrc {
public:
   __event void event1();   // C3724
};

[event_receiver(native)]
class CEventRec {
public:
   void handler1() {
   }

   void HookEvents(CEventSrc* pSrc) {
      __hook(CEventSrc::event1, pSrc, CEventRec::handler1);
   }

   void UnhookEvents(CEventSrc* pSrc) {
      __unhook(CEventSrc::event1, pSrc, CEventRec::handler1);
   }
};

int main() {
}
```
