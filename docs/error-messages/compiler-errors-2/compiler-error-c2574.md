Nem destruidores nem construtores podem ser declarados **`static`** .

O exemplo a seguir gera C2574:

```cpp
// C2574.cpp
// compile with: /c
class A {
   virtual static ~A();   // C2574
   //  try the following line instead
   // virtual ~A();
};
```
