# <a name="compiler-error-c2557"></a>Erro do compilador C2557

' identifier ': membros privados e protegidos não podem ser inicializados sem um construtor

Somente membros e amigos podem atribuir um valor a um membro privado ou protegido. Membros não públicos devem ser inicializados no construtor de classe.
