# <a name="expression-evaluator-error-cxx0023"></a>Erro CXX0023 (avaliador de expressão)

base inadequada

O avaliador de expressão C não reconhece a base especificada. Somente as radixs decimal, hexadecimal e octal são válidas.

Esse erro é idêntico a CAN0023.
