# <a name="expression-evaluator-error-cxx0010"></a>Erro CXX0010 (avaliador de expressão)

colchete esquerdo ausente

A expressão contém colchetes desbalanceos. Digite novamente a expressão com colchetes balanceados.

Esse erro é idêntico a CAN0010.
