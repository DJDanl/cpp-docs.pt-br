# <a name="resource-compiler-fatal-error-rw1030"></a>Erro fatal RW1030 (compilador de recurso)

Erro de saída

Esse erro pode ser causado se o compilador de recursos não pôde criar o arquivo de destino. Verifique se há espaço em disco suficiente e se você tem permissão de gravação no volume.
