# <a name="resource-compiler-fatal-error-rc1109"></a>Erro fatal RC1109 (compilador de recurso)

erro ao criar o nome-do-recurso

Não foi possível criar o arquivo. res especificado. Verifique se ele não está sendo criado em uma unidade somente leitura. Use a opção/V para descobrir se o arquivo está sendo criado.
