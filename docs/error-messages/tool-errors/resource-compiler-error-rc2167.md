# <a name="resource-compiler-error-rc2167"></a>Erro RC2167 (compilador de recurso)

campo VERSIONINFO não reconhecido; INÍCIO ou vírgula esperado

Um campo não reconhecido foi encontrado na parte **fixa** de uma declaração de estrutura **VERSIONINFO** . Um campo **VERSIONINFO** deve ser DWORDS separados por uma vírgula.
