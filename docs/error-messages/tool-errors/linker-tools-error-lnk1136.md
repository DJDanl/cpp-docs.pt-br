# <a name="linker-tools-error-lnk1136"></a>Erro das Ferramentas de Vinculador LNK1136

arquivo inválido ou corrompido

O arquivo de entrada tem um cabeçalho corrompido ou tem tamanho zero ou anormalmente pequeno.

LNK1136 também pode ser causado se o arquivo tiver sido especificado incorretamente como uma dependência adicional.
