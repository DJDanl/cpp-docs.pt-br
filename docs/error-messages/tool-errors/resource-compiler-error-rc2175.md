# <a name="resource-compiler-error-rc2175"></a>Erro RC2175 (compilador de recurso)

o nome do arquivo de recurso não está no formato 3, 0

O recurso especificado usou um formato anterior à versão 3, 0. O arquivo de recurso deve ser convertido ou recriado usando o formato para a versão 3, 0 ou posterior.
