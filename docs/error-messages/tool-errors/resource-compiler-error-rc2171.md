# <a name="resource-compiler-error-rc2171"></a>Erro RC2171 (compilador de recurso)

formato de cabeçalho DIB desconhecido

O cabeçalho de bitmap não é uma estrutura **BITMAPCOREHEADER** ou **BITMAPINFOHEADER** .
