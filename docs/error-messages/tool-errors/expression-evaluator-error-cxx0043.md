# <a name="expression-evaluator-error-cxx0043"></a>Erro CXX0043 (avaliador de expressão)

este ponteiro é usado fora da função membro

O **`this`** ponteiro só pode ser usado para funções de membros não estáticos.

Esse erro é idêntico a CAN0043.
