# <a name="command-line-warning-d9040"></a>Aviso D9040 (linha de comando)

ignorando a opção '/ANALYZE '; Os avisos de análise de código não estão disponíveis nesta edição do compilador

A opção de linha de comando **/Analyze** não está disponível em todas as edições do Visual Studio. Para corrigir esse aviso, alterne para uma edição com suporte do Visual Studio ou remova a opção de linha de comando.
