# <a name="resource-compiler-error-rc2163"></a>Erro RC2163 (compilador de recurso)

tipo de acelerador necessário [ASCII ou VIRTKEY]

O campo `type` na instrução **Accelerators** deve conter o valor ASCII ou VIRTKEY.
