# <a name="expression-evaluator-error-cxx0032"></a>Erro CXX0032 (avaliador de expressão)

dividir por 0

A expressão contém um divisor de zero, que é inválido. Esse divisor pode ser o número literal zero ou pode ser uma expressão que é avaliada como zero.

Esse erro é idêntico a CAN0032.
