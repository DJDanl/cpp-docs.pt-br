# <a name="nmake-fatal-error-u1097"></a>Erro fatal U1097 (NMAKE)

sintaxe de partes de nome de arquivo requer dependente

A dependência atual não tem um dependente explícito ou implícito. A sintaxe de partes do nome de arquivo, que usa o especificador de porcentagem (`%`), representa os componentes do primeiro dependente do destino atual.
