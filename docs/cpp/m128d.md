# <a name="__m128d"></a>__m128d

**Seção específica da Microsoft**

O tipo de dados **__m128d** para uso com as instruções de Streaming SIMD Extensions 2 são intrínsecas, é definido em \<emmintrin. h >.

```cpp
// data_types__m128d.cpp
#include <emmintrin.h>
int main() {
   __m128d x;
}
```

## <a name="remarks"></a>Comentários

Você não deve acessar os campos de **__m128d** diretamente. No entanto, você pode consultar esses tipos no depurador. Uma variável do tipo **__m128** é mapeada para os registros de XMM [0-7].

Variáveis do tipo **_m128d** são alinhadas automaticamente em limites de 16 bytes.

Não há suporte para o tipo de dados **__m128d** em processadores ARM.

**Fim da seção específica da Microsoft**
