# <a name="using-cricheditctrl"></a>Usando CRichEditCtrl

Um controle de edição rico é uma janela na qual o usuário pode inserir e editar texto. O texto pode ser atribuído à formatação de caractere e de parágrafo e pode incluir objetos OLE incorporados. O controle rich edit é representado em MFC pela classe [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) .
