# <a name="cmfcbasetabctrl-class"></a>Classe CMFCBaseTabCtrl

Implementa a funcionalidade base para janelas com guias.
Para obter mais detalhes, consulte o código-fonte localizado na pasta **vc\\atlmfc\\src\\MFC** da instalação do Visual Studio.

## <a name="syntax"></a>Sintaxe

```
class CMFCBaseTabCtrl : public CWnd
```

## <a name="remarks"></a>Comentários

A `CMFCBaseTabCtrl` classe é uma classe abstrata. Portanto, ele não pode ser instanciado. Para criar uma janela com guias, você deve derivar uma classe `CMFCBaseTabCtrl`de. A biblioteca MFC contém alguns exemplos de classe derivada, dois dos quais são [Classe CMFCTabCtrl](../../mfc/reference/cmfctabctrl-class.md) e [Classe CMFCOutlookBarTabCtrl](../../mfc/reference/cmfcoutlookbartabctrl-class.md).

A partir do Visual Studio 2015, essa classe dá suporte ao Microsoft Acessibilidade Ativa.

## <a name="customization-tips"></a>Dicas de personalização

As seguintes dicas pertencem ao e `CMFCBaseTabCtrl Class` a todas as classes que herdam dele:

- Se você habilitar as guias desanexáveis, não mantenha os ponteiros para as janelas com guias. Essas guias desanexáveis podem ser criadas e destruídas dinamicamente. Portanto, os ponteiros podem se tornar inválidos.

- Você pode configurar o controle guia para que os usuários possam mover guias dinamicamente em um controle guia usando o mouse. Essa funcionalidade é incorporada à `CMFCBaseTabCtrl` classe. Para habilitá-lo, chame [CMFCBaseTabCtrl:: EnableTabSwap](#enabletabswap).

- Por padrão, as guias podem ser desanexadas quando você as adiciona a um controle guia. Você também pode adicionar guias não desanexáveis usando [CMFCBaseTabCtrl:: AddTab](#addtab). Se você definir o parâmetro `bDetachable` como `FALSE`, a guia não será desanexável. Você também pode alterar se as guias podem ser desanexadas chamando o método [CMFCBaseTabCtrl:: EnableTabDetach](#enabletabdetach).

- Os objetos derivados da [classe CWnd](../../mfc/reference/cwnd-class.md) podem ser colocados em uma barra de controle encaixáveis ou na guia encaixáveis. Para que todo o controle seja encaixado, você deve tornar `CWnd` o objeto encaixáveis. Para fazer isso, o MFC usa uma classe wrapper. Essa classe de wrapper é a [Classe CDockablePaneAdapter](../../mfc/reference/cdockablepaneadapter-class.md). Todos `CWnd` os objetos adicionados a uma barra de controle encaixáveis ou à guia encaixáveis serão encapsulados dentro `CDockablePaneAdapter` de um objeto. Você pode desabilitar o encapsulamento automático definindo o parâmetro `m_bEnableWrapping` do seu `CMFCBaseTablCtrl` objeto como `FALSE`. Você também pode alterar a classe que seu aplicativo usará como um wrapper usando o método [CMFCBaseTabCtrl:: SetDockingBarWrapperRTC](#setdockingbarwrapperrtc).
