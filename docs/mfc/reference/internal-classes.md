# Classes internas
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

As classes a seguir so usadas internamente em MFC.  Para abrangncia, esta seo descreve essas classes internas, mas elas no se destinam a serem usadas diretamente no seu cdigo.  
  
## Nesta seo  
  
|Classe|Descrio|  
|------------|---------------|  
|`CCommandManager Class`|A classe de `CCommandManager` gerencia comandos e sua associao com imagens.|  
|`CDialogImpl Class (MFC)`|Fornece detalhes de implementao para classes \- base na caixa de dilogo, como `CDialogEx`.|  
|`CDocumentAdapter Class`|A interface de IDocument das ferramentas necessrias para Pesquisar e manipuladores organiza.|  
|`CFrameImpl Class`|A personalizao da barra de ferramentas dos identificadores da classe de `CFrameImpl` para a prxima janela do quadro classifica: [Classe de CFrameWndEx](../../mfc/reference/cframewndex-class.md), [Classe CMDIFrameWndEx](../Topic/CMDIFrameWndEx%20Class.md), e [Classe COleIPFrameWndEx](../../mfc/reference/coleipframewndex-class.md).|  
|`CFullScreenImpl Class`|Comuns da funcionalidade de tela cheia das ferramentas para aplicativos de SDI e de documentos.|  
|`CMDIClientAreaWnd Class`|A classe  uma classe de `CMDIClientAreaWnd` auxiliar que simplifica a implementao de guias MDI e grupos com guia documentos.|  
|`CMemDC Class`|Uma classe de auxiliar para um contexto do dispositivo de memria.  O oferece suporte o contexto do dispositivo de memria que desenha offscreen.|  
|`CMenuHash Class`|L o estado de objetos de `CMFCToolBar` de um arquivo morto ou escrever o estado como um arquivo morto.|  
|`CMenuImages Class`|Fornece a funcionalidade para as imagens predefinidas exibio como os prximos botes, maximizar os botes, os botes de opo, e as setas.|  
|`CMFCCaptionButtonEx Class`|Mantm a rea de cliente no dos botes da legenda do sistema.|  
|`CMFCCaptionMenuButton Class`|Representa um boto no menu que est localizado na rea da legenda de um painel de tarefas ou de uma barra de ferramentas.|  
|`CMFCColorPropertySheet Class`|Usado por [Classe de CMFCColorDialog](../../mfc/reference/cmfccolordialog-class.md) para fornecer uma caixa de dilogo tabulada.|  
|`CMFCControlBarImpl Class`|Fornece detalhes de implementao para classes barra e baseadas controle, como `CMFCReBar` e `CMFCToolBar`.|  
|`CMFCControlRenderer Class`|Uma classe de auxiliar que trata a renderizao de imagem.|
