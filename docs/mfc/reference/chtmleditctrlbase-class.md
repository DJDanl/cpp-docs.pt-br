## <a name="remarks"></a>Comentários

`CHtmlEditCtrlBase`fornece funções de membro para os comandos de edição HTML do WebBrowser, como [Bold](#bold). (Alternativamente, você pode chamar [o ExecCommand](#execcommand) para executar o comando IDM_BOLD.)

`CHtmlEditCtrlBase`não se destina a ficar por conta própria. Ele foi projetado para ser uma classe base para classes derivadas que expõem a funcionalidade de edição HTML do WebBrowser (ver [CHtmlEditCtrl](../../mfc/reference/chtmleditctrl-class.md) e [CHtmlEditView](../../mfc/reference/chtmleditview-class.md)).

## <a name="chtmleditctrlbaseaddtoglyphtable"></a><a name="addtoglyphtable"></a>CHtmlEditCtrlBase::AddToGlyphTable

Adiciona uma entrada à tabela glifo, que especifica imagens a serem exibidas para tags específicas no modo de design.

```
HRESULT AddToGlyphTable(
    LPCTSTR szTag,
    LPCTSTR szImgUrl,
    unsigned short nTagType,
    unsigned short nAlignment,
    unsigned short nPosInfo,
    unsigned short nDirection,
    unsigned int nImgWidth,
    unsigned int nImgHeight) const;
```

### <a name="parameters"></a>Parâmetros

*szTag*<br/>
O nome da tag (por exemplo, "P" ou "tabela").

*szImgUrl*<br/>
A URL da imagem.

*nTagType*<br/>
Tipo de tag: 0 significa que a imagem é apenas para a tag de abertura. 1 significa que a imagem é apenas para a tag de fechamento. 2 significa que a imagem é para as tags de abertura e fechamento. Tags individuais como br e comentário devem ser adicionadas com o tipo de tag definido como 0.

*nAlinhamento*<br/>
Alinhamento (apenas elementos retangulares): Este parâmetro indica que a imagem é para um elemento com um atributo de alinhamento. Esquerda = 0, centro = 1, direita = 2 e indefinida = 3. Os atributos esquerdo, direito ou central devem ser explicitamente definidos no elemento.

*nPosInfo*<br/>
Informações de posicionamento. Determina qual o valor de posicionamento das folhas de estilo em cascata (CSS) que o glifo aplica, onde o posicionamento estático = 0, posicionamento absoluto = 1, posicionamento relativo = 2, e todos = 3. Este campo permite que você especifique um glifo para uma tag quando ela não estiver posicionada e outro glifo para mostrar um ponto de âncora quando a tag estiver posicionada.

*nDirecionamento*<br/>
A direção. Este parâmetro especifica a imagem de uma tag com base na ordem de leitura do idioma atual. 0 especifica da esquerda para a direita, 1 especifica da direita para a esquerda, 2 especifica de cima para baixo, 3 especifica de baixo para cima e 4 especifica tudo. Você normalmente define este campo para 4.

*nImgWidth*<br/>
A largura da imagem em pixels.

*nImgHeight*<br/>
A altura da imagem em pixels.

Retorna S_OK no sucesso, ou um erro HRESULT no fracasso.

### <a name="remarks"></a>Comentários
