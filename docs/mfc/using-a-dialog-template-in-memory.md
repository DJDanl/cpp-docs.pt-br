# <a name="using-a-dialog-template-in-memory"></a>Usando um modelo da caixa de diálogo na memória

Em vez de usar os métodos fornecidos na [tabela de criação de diálogo](../mfc/creating-a-dialog-class-with-code-wizards.md), você pode criar qualquer tipo de caixa de diálogo indiretamente de um modelo de caixa de diálogo na memória. Para obter mais informações, consulte Class [CDialog](../mfc/reference/cdialog-class.md) na *referência do MFC*.
