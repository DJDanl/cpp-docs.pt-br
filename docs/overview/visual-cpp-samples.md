- [Amostras do Microsoft Docs - C++](https://docs.microsoft.com/samples/browse/?term=c%2B%2B)

- [Amostras do Windows no GitHub](https://microsoft.github.io/windows/)

- [Amostras de código do Windows Dev Center](https://developer.microsoft.com/windows/samples)

- [Arquivo de amostra CodePlex](https://archive.codeplex.com/)

- [Exemplos de desenvolvimento de hardware para Windows](https://docs.microsoft.com/samples/browse/?products=windows-wdk)

## <a name="archived-c-samples-on-github"></a>Amostras C++ arquivadas no GitHub

O Visual Studio incluiu o código de amostra C++ em versões anteriores. O código de amostra foi instalado no Visual Studio, ou estava disponível como download separado.
