# <a name="masm-expressions-in-inline-assembly"></a>Expressões MASM em assembly embutido

**Seção específica da Microsoft**

O código de assembly embutido pode usar qualquer expressão MASM, que é qualquer combinação de operandos e operadores que é avaliada como um único valor ou endereço.
