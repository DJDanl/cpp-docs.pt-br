# <a name="end"></a>END

Marca o final de um módulo e, opcionalmente, define o ponto de entrada do programa para *procid*.

## <a name="syntax"></a>Sintaxe

> **Encerrar** ⟦*procid*⟧

## <a name="remarks"></a>Comentários

O argumento *procid* é válido somente no MASM de 32 bits.
