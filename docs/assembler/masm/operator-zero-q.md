# <a name="operator-zero"></a>operador ZERO?

Status do sinalizador zero. Usado somente no [. SE](dot-if.md), [. WHILE](dot-while.md), ou [. Repita](dot-repeat.md) os blocos e avaliados em tempo de execução, não no momento do assembly.
