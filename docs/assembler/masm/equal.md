# <a name=""></a>=

Atribui o valor numérico da *expressão* ao *nome*.

## <a name="syntax"></a>Sintaxe

> *expressão* de **=** de *nome*

## <a name="remarks"></a>Comentários

O símbolo pode ser redefinido mais tarde.
