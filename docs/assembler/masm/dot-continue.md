# <a name="continue-32-bit-masm"></a>. CONTINUAR (MASM-bit de 32 bits)

Gera código para saltar para a parte superior de um [. WHILE](dot-while.md) ou [. REPETIR](dot-repeat.md) bloco se a *condição* for verdadeira. (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> **. CONTINUE** ⟦ **.**  *Condição*If ⟧
