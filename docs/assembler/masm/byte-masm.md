# <a name="byte"></a>BYTE

Aloca e, opcionalmente, Inicializa um byte de armazenamento para cada *inicializador*.

## <a name="syntax"></a>Sintaxe

> ⟦*Name*⟧ **byte** *inicializador* ⟦ __,__ *inicializador* ... ⟧

## <a name="remarks"></a>Comentários

Também pode ser usado como um especificador de tipo em qualquer lugar em que um tipo é válido.
