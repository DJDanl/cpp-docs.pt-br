# <a name=""></a>\@\@:

Define um rótulo de código reconhecível somente entre *Label1* e *Label2*, em que *Label1* é o início do código ou o rótulo de `@@:` anterior e *Label2* é o final do código ou o próximo rótulo de `@@:`. Consulte [\@B](at-b.md) e [\@F](at-f.md).

## <a name="syntax"></a>Sintaxe

> **\@\@:**
