# <a name="mmx-32-bit-masm"></a>. MMX (MASM-bit de 32 bits)

Habilita o assembly de instruções MMX ou de instrução única (SIMD). (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> **.MMX**
