# <a name="equ"></a>EQU

A primeira diretiva atribui o valor numérico da *expressão* ao *nome*.

## <a name="syntax"></a>Sintaxe

> *expressão* de **equ** de *nome*
>
> *nome* **EQU** __\<__ *texto* __>__

## <a name="remarks"></a>Comentários

O *nome* não pode ser redefinido mais tarde.

A segunda diretiva atribui o *texto* especificado ao *nome*. O *nome* pode ser atribuído a um *texto* diferente posteriormente. Consulte [TEXTEQU](textequ.md).
