# <a name="operator-opattr"></a>operador OPATTR

Retorna uma palavra que define o modo e o escopo da *expressão*. O byte baixo é idêntico ao byte retornado por [. Digite](operator-dot-type.md). O byte alto contém informações adicionais.

## <a name="syntax"></a>Sintaxe

> *Expressão* OPATTR
