# <a name="operator-highword"></a>operador HIGHWORD

Retorna os 16 bits altos de 32 bits de *expressão*baixa. As expressões MASM são valores de 64 bits.

## <a name="syntax"></a>Sintaxe

> *Expressão* HIGHWORD
