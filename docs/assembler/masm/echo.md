# <a name="echo"></a>ECHO

Exibe a *mensagem* para o dispositivo de saída padrão (por padrão, a tela).

## <a name="syntax"></a>Sintaxe

> *Mensagem* de eco

## <a name="remarks"></a>Comentários

O mesmo que [% out](../../assembler/masm/percent-out.md).
