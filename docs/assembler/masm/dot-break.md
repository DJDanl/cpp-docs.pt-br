# <a name="break-32-bit-masm"></a>. INTERRUPÇÃO (MASM de 32 bits)

Gera o código para encerrar um [. WHILE](dot-while.md) ou [. REPETIR](dot-repeat.md) bloco se a *condição* for verdadeira. (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> **. INTERROMPER** ⟦ **.**  *Condição*If ⟧
