# <a name="while"></a>WHILE

Repete o assembly de *instruções* de bloqueio, contanto que a *constante* permaneça verdadeira.

## <a name="syntax"></a>Sintaxe

> **Embora** a *constante*\
> *instruções*\
> **ENDM**
