# <a name="operator-"></a>Operador ;;

Trata o *texto* como um comentário em uma macro que aparece apenas na definição de macro. A listagem não mostra o *texto* onde a macro é expandida.

## <a name="syntax"></a>Sintaxe

> **;;** *texto* de
