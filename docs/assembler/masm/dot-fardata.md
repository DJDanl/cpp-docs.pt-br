# <a name="fardata-32-bit-masm"></a>. FARDATA (MASM de 32 bits)

Quando usado com [. MODELO](dot-model.md), inicia um segmento de dados distante para dados inicializados (nome do segmento FAR_DATA ou *nome*). (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> **.** *Nome*do FARDATA ⟦ ⟧
