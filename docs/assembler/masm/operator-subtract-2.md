# <a name="operator--"></a>Operador -

Retorna *expression1* menos *expression2*. O segundo operador reverte o sinal de *expressão*.

## <a name="syntax"></a>Sintaxe

> *expression1* **-** *expression2*
>
> *expressão* de-
