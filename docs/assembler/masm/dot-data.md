# <a name="data"></a>.DATA

 (somente MASM de 32 bits.) Quando usado com [. MODELO](dot-model.md), inicia um segmento de dados Near para dados inicializados (nome do segmento _DATA).

## <a name="syntax"></a>Sintaxe

> **.** \ de dados
> ⟦ *segmentItem* ⟧...
