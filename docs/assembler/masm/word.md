# <a name="word"></a>WORD

Aloca e, opcionalmente, Inicializa uma palavra (2 bytes) de armazenamento para cada *inicializador*. Também pode ser usado como um especificador de tipo em qualquer lugar em que um tipo é válido. O [DW](dw.md) é um sinônimo do **Word**.

## <a name="syntax"></a>Sintaxe

> ⟦*Name*⟧ **Word** *inicializador* ⟦ __,__ *inicializador* ... ⟧
