# <a name="operator-"></a>Operador ||

OR lógico. Usado somente no [. SE](dot-if.md), [. WHILE](dot-while.md), ou [. Repita](dot-repeat.md) os blocos e avaliados em tempo de execução, não no momento do assembly.

## <a name="syntax"></a>Sintaxe

> *expression1* **||** *expression2*
