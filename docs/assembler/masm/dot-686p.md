# <a name="686p-32-bit-masm"></a>.686P (MASM de 32 bits)

Habilita o assembly de todas as instruções (incluindo privilégios) para o processador Pentium Pro. (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> **.686P**
