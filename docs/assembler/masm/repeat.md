# <a name="repeat"></a>REPETIR

Marca um bloco que deve ser repetido vezes na *expressão* . O mesmo que [REPT](rept.md).

## <a name="syntax"></a>Sintaxe

> \ de *expressão* de **repetição**
> *instruções*\
> **ENDM**
