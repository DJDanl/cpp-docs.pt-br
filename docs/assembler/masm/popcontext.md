# <a name="popcontext"></a>POPCONTEXT

Restaura parte ou todo o *contexto* atual (salvo pela diretiva [PUSHCONTEXT](pushcontext.md) ). O *contexto* pode ser **assumido** (somente em massa de 32 bits), **Radix**, **listagem**, **CPU** (somente em massa de 32 bits) ou **todos**.

## <a name="syntax"></a>Sintaxe

> *Contexto* de POPCONTEXT
