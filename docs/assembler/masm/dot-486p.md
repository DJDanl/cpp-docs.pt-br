# <a name="486p-32-bit-masm"></a>.486P (MASM de 32 bits)

Habilita o assembly de todas as instruções (incluindo privilégios) para o processador 80486. (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> **.486P**
