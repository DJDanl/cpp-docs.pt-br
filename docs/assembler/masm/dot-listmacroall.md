# <a name="listmacroall"></a>.LISTMACROALL

## <a name="syntax"></a>Sintaxe

> **.LISTMACROALL**

## <a name="remarks"></a>Comentários

O mesmo que [. LALL](../../assembler/masm/dot-lall.md).
