# <a name="operator-lt-masm-run-time"></a>operador &lt;= (tempo de execução MASM)

É menor ou igual a. Usado somente no [. SE](dot-if.md), [. WHILE](dot-while.md), ou [. Repita](dot-repeat.md) os blocos e avaliados em tempo de execução, não no momento do assembly.
