# <a name="real10"></a>REAL10

Aloca e, opcionalmente, Inicializa um número de ponto flutuante de 10 bytes para cada *inicializador*.

## <a name="syntax"></a>Sintaxe

> *nome* **REAL10** *inicializador* ⟦ __,__ *inicializador* ... ⟧
