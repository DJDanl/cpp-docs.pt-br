# <a name="ends"></a>ENDS

Marca o fim do segmento, a estrutura ou o *nome* da União iniciado anteriormente com o [segmento](segment.md), [struct](struct-masm.md), [União](union.md)ou uma diretiva de segmento simplificado.

## <a name="syntax"></a>Sintaxe

> *nome* **termina**
