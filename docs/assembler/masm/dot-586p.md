# <a name="586p-32-bit-masm"></a>.586P (MASM de 32 bits)

Habilita o assembly de todas as instruções (incluindo privilégios) para o processador Pentium. (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> **.586P**
