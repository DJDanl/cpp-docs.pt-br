# <a name="sizestr"></a>\@SizeStr

Uma função de macro que retorna o comprimento da cadeia de caracteres especificada. Retorna um número inteiro.

## <a name="syntax"></a>Sintaxe

> **\@SizeStr (cadeia de** *caracteres* **)**
