# <a name="qword"></a>QWORD

Aloca e, opcionalmente, inicializa 8 bytes de armazenamento para cada *inicializador*. Também pode ser usado como um especificador de tipo em qualquer lugar em que um tipo é válido. **QWORD** é um sinônimo de [DQ](dq.md).

## <a name="syntax"></a>Sintaxe

> ⟦*Name*⟧ **QWORD** *inicializador* ⟦ __,__ *inicializador* ... ⟧
