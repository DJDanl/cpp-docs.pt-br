# <a name="ifidn-ifidni"></a>IFIDN, IFIDNI

Concede o assembly se os itens de texto forem idênticos.

## <a name="syntax"></a>Sintaxe

> **IFIDN** *textitem1* __,__ *textitem2*\
> **IFIDNI** *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Comentários

Se **IFIDNI** for fornecido, a comparação não diferenciará maiúsculas de minúsculas. Confira [se](if-masm.md) para obter a sintaxe completa.
