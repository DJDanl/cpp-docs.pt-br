# <a name="errdef"></a>.ERRDEF

Gera um erro se o *nome* for um rótulo, uma variável ou um símbolo definido anteriormente.

## <a name="syntax"></a>Sintaxe

> **. ERRDEF** *nome* ⟦ __,__ *mensagem*⟧
