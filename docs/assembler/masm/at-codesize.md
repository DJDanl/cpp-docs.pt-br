# <a name="codesize"></a>Códigos de \@

0 para **modelos minúsculos**, **pequenos**, **compactados**e **simples** e 1 para modelos **médios**, **grandes**e **enormes** (equivalente numérico).

## <a name="syntax"></a>Sintaxe

> **\@CodeSize**
