# <a name="operator-gt-masm-run-time"></a>operador &gt;= (tempo de execução MASM)

É maior ou igual a. Usado somente no [. SE](../../assembler/masm/dot-if.md), [. WHILE](../../assembler/masm/dot-while.md), ou [. Repita](../../assembler/masm/dot-repeat.md) os blocos e avaliados em tempo de execução, não no momento do assembly.

## <a name="syntax"></a>Sintaxe

> *expression1* **>=** *expression2*
