# <a name="while-32-bit-masm"></a>. WHILE (MASM de 32 bits)

Gera um código que executa o bloco de *instruções* enquanto a *condição* permanece verdadeira. (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> **. ENQUANTO** a *condição*\
> *instruções*\
> **.ENDW**
