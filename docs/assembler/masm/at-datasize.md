# <a name="datasize"></a>\@de datas

0 para **modelos minúsculos**, **pequenos**, **médios**e **simples** , 1 para modelos **compactados** e **grandes** , e 2 para modelo **grande** (equivalente numérico).

## <a name="syntax"></a>Sintaxe

> **\@de datas**
