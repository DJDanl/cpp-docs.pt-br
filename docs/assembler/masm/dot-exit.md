# <a name="exit-32-bit-masm"></a>. EXIT (MASM-bit de 32)

Gera o código de encerramento. (somente MASM de 32 bits.)

## <a name="syntax"></a>Sintaxe

> . EXIT [[expressão]]

## <a name="remarks"></a>Comentários

Retorna a *expressão* opcional para o Shell.
