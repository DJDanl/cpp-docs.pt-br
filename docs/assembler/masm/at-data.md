# <a name="data"></a>\@dados

O nome do grupo de dados padrão. Avalia como DGROUP para todos os modelos, exceto **Flat**. Avalia para **plano** no modelo de memória **simples** (macro de texto).

## <a name="syntax"></a>Sintaxe

> **\@dados**
