# <a name="oword"></a>OWORD

Usado como um especificador de tipo quando um tipo de dados de 16 bytes é necessário.

## <a name="syntax"></a>Sintaxe

> ⟦*Name*⟧ **OWORD**  *inicializador* ⟦ __,__ *inicializador* ... ⟧
