# <a name="typedef"></a>TYPEDEF

Define um novo tipo ou **proto** chamado *Name*, que é equivalente a *Type* ou *protoDefinition*.

## <a name="syntax"></a>Sintaxe

> *tipo* de **TYPEDEF** de *nome* | **proto** *protoDefinition*
