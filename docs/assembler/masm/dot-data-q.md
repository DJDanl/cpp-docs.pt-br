# <a name="data"></a>.DATA?

(somente MASM de 32 bits.) Quando usado com [. MODELO](dot-model.md), inicia um segmento de dados próximo para dados não inicializados (nome do segmento _BSS).

## <a name="syntax"></a>Sintaxe

> **. DADOS?** \
> ⟦ *segmentItem* ⟧...
