# <a name="elseif"></a>ELSEIF

Combina [else](else-masm.md) e [If](if-masm.md) em uma instrução.

## <a name="syntax"></a>Sintaxe

> *Constante* de ElseIf\
> *instruções*\
> ⟦**ELSE**\
> *else – instruções*⟧ \
> **ENDIF**

Consulte [se](if-masm.md) para obter mais informações.
