# <a name="operator-dup"></a>operador DUP

Especifica o número de *contagem* de declarações de *inicialvalue*.

## <a name="syntax"></a>Sintaxe

> *contagem de* **dup** __(__ *inicialvalue* ⟦ __,__ *inicialvalue* ... ⟧ __)__
