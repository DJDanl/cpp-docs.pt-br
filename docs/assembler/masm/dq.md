# <a name="dq"></a>DQ

Aloca e, opcionalmente, inicializa 8 bytes de armazenamento para cada *inicializador*. Também pode ser usado como um especificador de tipo em qualquer lugar em que um tipo é válido. O **DQ** é um sinônimo de [QWORD](qword.md).

## <a name="syntax"></a>Sintaxe

> ⟦*Name*⟧ **DQ** *inicializador* ⟦ __,__ *inicializador* ... ⟧
