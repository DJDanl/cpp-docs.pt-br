# <a name="catstr"></a>\@CatStr

Função de macro que concatena uma ou mais cadeias de caracteres. Retorna uma cadeia de caracteres.

## <a name="syntax"></a>Sintaxe

> **\@CatStr (** *seqüência1* ⟦ __,__ *seqüência2*... ⟧ **)**
