# <a name="catstr"></a>\@CatStr

Macro function that concatenates one or more strings. Retorna uma cadeia de caracteres.

## <a name="syntax"></a>Sintaxe

> **\@CatStr(** *string1* ⟦ __,__ *string2*...⟧ **)**
