# <a name="substr"></a>\@SubSt

Uma função de macro que retorna uma subcadeia de caracteres começando na *posição*.

## <a name="syntax"></a>Sintaxe

> **\@subst (** *cadeia de caracteres* __,__ *posição* ⟦ __,__ *comprimento*⟧ **)**
