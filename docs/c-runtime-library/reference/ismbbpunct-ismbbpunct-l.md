# <a name="_ismbbpunct-_ismbbpunct_l"></a>_ismbbpunct, _ismbbpunct_l

Determina se um dado caractere é um caractere de pontuação.

## <a name="syntax"></a>Sintaxe

```C
int _ismbbpunct(
   unsigned int c
);
int _ismbbpunct_l(
   unsigned int c,
   _locale_t locale
);
```

### <a name="parameters"></a>Parâmetros

*C*<br/>
Inteiro a ser testado.

*Localidade*<br/>
Localidade a usar.

## <a name="return-value"></a>Valor retornado

**_ismbbpunct** retorna um valor não zero se o inteiro *c* for um símbolo de pontuação não ASCII. **_ismbbpunct** usa o local atual para quaisquer configurações de caracteres dependentes de localidades. **_ismbbpunct_l** é idêntica, exceto que usa o local que passou. Para obter mais informações, consulte [Localidade](../../c-runtime-library/locale.md).

## <a name="remarks"></a>Comentários

Por padrão, o estado global desta função é escopo para o aplicativo. Para mudar isso, consulte [Estado Global no CRT](../global-state.md).
