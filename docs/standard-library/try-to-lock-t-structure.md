# <a name="try_to_lock_t-structure"></a>Estrutura try_to_lock_t

Representa um **`struct`** que define um objeto [try_to_lock](../standard-library/mutex-functions.md#try_to_lock) . Usado para selecionar um dos construtores sobrecarregados de [unique_lock](../standard-library/unique-lock-class.md).

## <a name="syntax"></a>Sintaxe

```cpp
struct try_to_lock_t;
```

## <a name="requirements"></a>Requisitos

**Cabeçalho:**\<mutex>

**Namespace:** std
