# <a name="ltunordered_setgt"></a>&lt;unordered_set&gt;

## <a name="requirements"></a>Requisitos

**Cabeçalho:**\<unordered_set>

**Namespace:** std

> [!NOTE]
> A \<unordered_set> biblioteca também usa a `#include <initializer_list>` instrução.
