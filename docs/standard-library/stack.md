# <a name="ltstackgt"></a>&lt;stack&gt;

Define a pilha de modelos de classe e dois modelos de suporte.

## <a name="requirements"></a>Requisitos

**Cabeçalho:** \<stack>

**Namespace:** std

> [!NOTE]
> A biblioteca \<stack > também usa a instrução `#include <initializer_list>`.
