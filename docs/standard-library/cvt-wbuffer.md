# <a name="ltcvtwbuffergt"></a>&lt;cvt/wbuffer&gt;

O cabeçalho `<cvt/wstring>` nas versões anteriores do Visual Studio definiu a classe template do modelo de classe [wbuffer_convert](../standard-library/wbuffer-convert-class.md) no namespace stdext:: CVT. O cabeçalho é mantido para compatibilidade com versões anteriores. O novo código deve usar a versão da classe que é definida em [\<locale>](../standard-library/locale.md) no namespace `std`

## <a name="syntax"></a>Sintaxe

```cpp
#include <cvt/wbuffer>
```
