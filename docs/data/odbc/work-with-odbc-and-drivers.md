# <a name="work-with-odbc-and-drivers"></a>Trabalhar com ODBC e drivers

O ODBC permite que você escreva aplicativos que não dependem de apenas um DBMS (sistema de gerenciamento de banco de dados). Você pode escrever um aplicativo que possa operar em vários DBMSs para os quais seus usuários têm o driver ODBC específico do DBMS correto.
