# <a name="schema-rowsets"></a>Conjuntos de linhas de esquema

O esquema (ou metadados) é qualquer informação sobre a estrutura ou organização de um banco de dados, como informações sobre o provedor, conjunto de linhas, tabela, colunas ou outras informações, exceto o conteúdo real do banco de dados. OLE DB permite que você obtenha essas informações sem abrir o conjunto de linhas usando conjuntos de itens de esquema.

Para obter informações sobre como usar conjuntos de linhas de esquema, consulte [obtendo metadados com conjuntos de linhas de esquema](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
