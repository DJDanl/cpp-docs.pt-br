# <a name="cmultipleresults-class"></a>Classe CMultipleResults

Se você quiser que um comando manipule vários conjuntos de resultados, use `CMultipleResults` para o argumento de modelo `CCommand` *TMultiple*.

## <a name="syntax"></a>Sintaxe

```cpp
class CMultipleResults
```

## <a name="remarks"></a>Comentários

Para lidar com vários conjuntos de resultados, [CCommand](../../data/oledb/ccommand-class.md) deve herdar dessa classe.

## <a name="requirements"></a>Requisitos

**Cabeçalho:** atldbcli
