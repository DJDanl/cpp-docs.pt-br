# <a name="improper_scheduler_detach-class"></a>Classe improper_scheduler_detach

Essa classe descreve uma exceção gerada quando o método `CurrentScheduler::Detach` é chamado em um contexto que não foi anexado a nenhum Agendador usando o método `Attach` de um objeto `Scheduler`.

## <a name="syntax"></a>Sintaxe

```cpp
class improper_scheduler_detach : public std::exception;
```

|----------|-----------------|
